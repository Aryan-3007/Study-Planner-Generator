# AI Study Planner

## Problem
Students struggle to prioritize subjects efficiently before exams, leading to last-minute stress and poor time management.

## Solution
An AI-powered study planning system that prioritizes subjects based on syllabus remaining, difficulty, and exam urgency. It generates adaptive daily study schedules and intelligent strategy recommendations.

## The logic on which the timetable get generated
- priority score = (Difficulty weight * syllabus remaining) + urgency multiplier
- Distributed study hours = priority / total priority * Daily study hours

## Features
- Priority-based subject allocation
- Urgency detection (exam tomorrow mode)
- Burnout risk detection
- Adaptive AI insights
- Revision strategy suggestions
